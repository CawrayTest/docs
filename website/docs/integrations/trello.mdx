Turn features On / Off right from a linked Card on your Trello board with . Also you can add Targeting or Percentage Rules just as easily.

The <a href="https://trello.com/power-ups/5e694b66d2511a3601ebd0fb" target="_blank">ConfigCat's Trello Power-Up</a> allows you to connect your Trello Cards and feature flags. Create or link existing flags to your Trello cards without leaving your Trello board.

Turn features On / Off right from a linked card on your Trello board. Modify or add new Targeting or Percentage Rules easily to the linked flags.

This guide will help you with the power-up installation and familiarise you with the plugin usage.

## Installation

1. Add <a href="https://trello.com/power-ups/5e694b66d2511a3601ebd0fb" target="_blank">ConfigCat Power-Up</a> to your board.
2. Select `Settings -> Authorize Account`.
3. Get your ConfigCat Public API credentials and copy to he inputs. To learn more, read about <a href="https://app.configcat.com/my-account/public-api-credentials" target="_blank">ConfigCat Public API credentials</a>.
4. Click authorize.

:::info
Every Trello user should authorize ConfigCat in Trello who wants to use the ConfigCat Feature Flags power-up.
:::

## Usage

### Creating new feature flags

1. Open any Card on your Trello board.
2. Push `Create and Link Feature Flag` button.
3. Select a Product, Config, where you want to create the feature flag.
4. Setup your feature flag.
5. Select which environment would you like to link to this item.
6. When linked, you can turn your features On / Off right from this Card.

<img src="/docs/assets/trello_use.gif" className="zoomable" alt="Usage of the ConfigCat Feature Flags Trello Power-Up" />

### Linking existing feature flags

1. Open any Card on your Trello board.
2. Push `Link Feature Flag` button.
3. Select a ConfigCat Product, Config, Environment and a Feature Flag to be linked to your issue.
4. When linked, you can turn your features On / Off right from this Issue.

<img src="/docs/assets/trello_use.gif" className="zoomable" alt="Usage of the ConfigCat Feature Flags Trello Power-Up" />

### View and Edit linked feature flags

1. Open an Card on your Trello board with linked feature flag.
2. You can see the linked feature flags in the `Feature Flag (ConfigCat)` section.
3. You can turn your features On / Off right from this Card.
4. You can add new Targeting Rules with User, Segment or Prerequisite conditions with large selection of Comparators.
5. You can add new Percentage Options to Feature Flags or Targeting Rules.
6. You can remove Targeting Rules as well.

<img src="/docs/assets/trello_use.gif" className="zoomable" alt="Usage of the ConfigCat Feature Flags Trello Power-Up" />

### Remove linked feature flags

1. Open a Card on your Trello board with linked feature flag.
2. Remove the linked feature flag by opening the "More menu" and selecting the "Remove from card" option.

<img src="/docs/assets/trello_use.gif" className="zoomable" alt="Usage of the ConfigCat Feature Flags Trello Power-Up" />
